Simple Flat Mapper (SFM) v2.13

SFM provides very fast and easy to use mapper for

  • Micro ORM
  • Csv Parser
  • Csv Mapper

It also has modules to provide integration with

  • Cassandra Datastax
  • jOOQ
  • POI Excel Spreadsheet
  • Query DSL
  • Spring Jdbc
  • Sql2o

The binaries are available in maven central. There is a build for

Java 8

        <dependency>
            <groupId>org.simpleflatmapper</groupId>
            <artifactId>sfm</artifactId>
            <version>2.13</version>
        </dependency>

Java 7

        <dependency>
            <groupId>org.simpleflatmapper</groupId>
            <artifactId>sfm</artifactId>
            <version>2.13</version>
            <classifier>jdk17</classifier>
        </dependency>

Java 6

        <dependency>
            <groupId>org.simpleflatmapper</groupId>
            <artifactId>sfm</artifactId>
            <version>2.13</version>
            <classifier>jdk16</classifier>
        </dependency>
